# OSS (Object Storage Service) - 对象存储

## 基本信息

- **产品全称**: Object Storage Service (OSS)
- **产品类别**: 存储服务
- **产品定位**: 完全托管的云对象存储服务

## 产品用途

OSS 用于存储和访问任意数量的任何类型数据，典型应用场景：

- 内容分发（图片、视频、软件下载）
- 数据备份与恢复
- 云端归档（冷数据存储）
- 数据湖存储基础
- 静态网站托管
- 网盘/在线文档存储
- IoT 设备数据存储

## 产品特点

- **高耐久性设计**: 12 个 9（99.9999999999%）数据耐久性
- **多层存储类型**: 标准存储、低频访问存储、归档存储、冷归档存储
- **S3 兼容**: API 与 Amazon S3 兼容，降低迁移门槛
- **内置媒体处理**: 图片处理（IMG）、视频截图等无需额外计算集群
- **传输加速**: 全球数据中心优化传输路径
- **生命周期管理**: 自动在不同存储层间迁移数据优化成本

## 产品优点

1. **领先 SLA**: 本地冗余可用性 99.99%，跨区域冗余可用性 99.995%
2. **成本优化**: 归档存储检索 1 分钟内完成，冷归档 1 小时内完成（竞品需数小时到 48 小时）
3. **免费额度**: API 请求、外网流出流量等均有一定免费额度
4. **合规认证**: 唯一通过 Cohasset Associates 审计认证的中国云存储服务
5. **生态集成**: 与 MaxCompute、EMR、Flink、PAI 等分析引擎深度集成

## 产品缺点

1. **小文件性能**: 海量小文件场景下管理和访问效率相对较低
2. **跨区域同步**: 跨区域复制存在延迟，不适合实时强一致性场景
3. **复杂权限配置**: Bucket Policy、ACL、RAM Role 组合配置较为复杂
4. **检索能力有限**: 对象存储本身不支持 SQL 查询，需依赖 DLA 等外部服务

## 适用场景

- **图片/视频存储与处理**: 电商、社交平台
- **备份归档**: 企业数据长期保存、合规要求
- **静态网站**: 无需服务器的前端应用托管
- **数据湖**: 大数据分析底层存储
