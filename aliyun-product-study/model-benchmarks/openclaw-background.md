# 龙虾（OpenClaw）背景与生态

## 一、什么是"龙虾"？

"龙虾"是开源 AI 智能体框架 **OpenClaw** 的网络昵称。

- **发布时间**: 2025 年 11 月
- **创始人**: Peter Steinberger（奥地利开发者）
- **核心定位**: 真正做事的 AI（AI that gets things done）

与传统的对话型 AI 不同，OpenClaw 能够直接操作用户电脑，包括：

| 能力 | 说明 |
|------|------|
| 读写文件 | 直接编辑、创建、删除本地文件 |
| 收发邮件 | 自动处理邮件收发任务 |
| 调用浏览器 | 操作浏览器完成网页交互自动化 |
| 终端执行 | 运行命令行、执行脚本 |

## 二、"百虾大战"背景

2026 年初，"龙虾热"席卷国内外，各大厂商纷纷入局，市场迎来"百虾大战"。

| 厂商 | 龙虾产品策略 | 核心抓手 |
|------|------------|---------|
| 阿里千问 | 开放模型，供开发者接入 OpenClaw | 模型能力（编程 + 多模态） |
| 火山引擎 | 自研 ArkClaw 平台，提供端到端服务 | 模型 + 安全 + Skills 生态三位一体 |

## 三、OpenClaw 协议兼容性

OpenClaw 兼容多种 API 协议，为第三方模型接入提供标准化接口：

| 协议 | 说明 | 支持模型 |
|------|------|---------|
| Claude 协议 | 兼容 Anthropic API 格式 | 千问 3.6 系列 |
| OpenAI 协议 | 兼容 OpenAI API 格式 | 千问系列、豆包系列 |

**重要动态**: OpenClaw 创始人 Peter Steinberger 已宣布，**下一代 OpenClaw 将率先提供阿里千问大模型的官方支持**，这是对千问模型在龙虾场景下能力的重要背书。

## 四、龙虾场景典型应用

| 应用场景 | 说明 | 适合模型 |
|---------|------|---------|
| 氛围编程 | 一句话生成可用的网站、游戏、小程序 | 千问 3.6-Plus |
| 播客速通 | 长音频核心内容提取 | 千问 3.5-Omni |
| 视频深度解析 | 逐帧分析视频内容 | 千问 3.5-Omni |
| 飞书日程管理 | 处理日程、文档、表格 | 火山 ArkClaw |
| 商品网页生成 | 基于图片生成完整商品网页 | 千问 3.6-Plus |
